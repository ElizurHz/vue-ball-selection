# vue-ball-selection

> A ball-style selection component made by Vue.js + vuex

> Can be used in many ways, for example, lottery

```
props: {
  title: {
    type: String,
    required: true
  },
  ballList: {
    type: Array,
    required: true
  },
  ballShortcutSelection: {
    type: Array,
    required: true
  }
}
```
